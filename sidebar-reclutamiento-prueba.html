Información del componente

Este sidebar está diseñado específicamente para el módulo de reclutamiento e incluye:

  • Dashboard - Vista general del módulo de reclutamiento
  • Vacantes - Gestión de vacantes y procesos de selección
  • Plantillas - Plantillas personalizadas de selección
  • Menú de perfil del usuario
  • Toggle de modo oscuro

Instrucciones de uso

Para usar este componente en otras páginas:

  1. Incluir el CSS: components-sidebar.css
  2. Incluir el JS: components/sidebar-reclutamiento.js
  3. Agregar el contenedor: <div id="sidebar-container"></div>
  4. Llamar la función: loadSidebarReclutamiento('boton-activo')
