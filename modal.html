UBITS Modal Component

Componente modal oficial del sistema de diseño UBITS

Descripción

El componente Modal de UBITS es un diálogo modal que aparece sobre el contenido principal para mostrar información importante, formularios o solicitar confirmación del usuario. Está diseñado siguiendo los principios del sistema de diseño UBITS con tokens de color, tipografía y espaciado oficiales.

Características

  • ✅ Diseño basado en tokens UBITS oficiales
  • ✅ Responsive y adaptable a móviles
  • ✅ Animaciones suaves de entrada y salida
  • ✅ Soporte para backdrop blur
  • ✅ Cierre con ESC y click en backdrop
  • ✅ Botones UBITS integrados
  • ✅ Formularios y validación
  • ✅ Scroll interno cuando es necesario

Implementación Básica

Para usar el componente Modal, incluye los archivos CSS y JS necesarios:

<!-- CSS requerido --> <link rel="stylesheet" href="ubits-colors.css"> <link rel="stylesheet" href="ubits-typography.css"> <link rel="stylesheet" href="components/button.css"> <link rel="stylesheet" href="components/modal.css"> <link rel="stylesheet" href="fontawesome-icons.css"> <!-- JavaScript requerido --> <script src="components/modal.js"></script>

Ejemplos de Uso

Modal Básico

Modal simple con título, contenido y botones personalizados.

const modal = UBITSModalManager.create({ title: 'Mi Modal', content: '<p>Contenido del modal</p>', footerRight: [ { text: 'Cancelar', variant: 'secondary', onclick: 'UBITSModalManager.close("' + modal.id + '")' }, { text: 'Aceptar', variant: 'primary', onclick: 'UBITSModalManager.close("' + modal.id + '")' } ] }); modal.open();

Modal de Confirmación

Modal para confirmar acciones importantes.

showConfirmModal({ title: 'Eliminar plantilla', message: '¿Estás seguro de que quieres eliminar esta plantilla? Esta acción no se puede deshacer.', confirmText: 'Eliminar', cancelText: 'Cancelar', variant: 'primary', onConfirm: () => { console.log('Acción confirmada'); // Lógica de eliminación }, onCancel: () => { console.log('Acción cancelada'); } });

Modal de Formulario

Modal con formulario para crear o editar datos.

const formHTML = ` <div class="ubits-modal__form-group"> <label class="ubits-modal__label">Nombre de la plantilla</label> <input type="text" class="ubits-modal__input" placeholder="Ej: Flujo de contratación"> </div> <div class="ubits-modal__form-group"> <label class="ubits-modal__label">Categoría</label> <select class="ubits-modal__select"> <option value="">Selecciona una categoría</option> <option value="contratacion">Contratación</option> <option value="evaluacion">Evaluación</option> </select> </div> `; showFormModal({ title: 'Crear nueva plantilla', formHTML: formHTML, submitText: 'Crear', cancelText: 'Cancelar', onSubmit: () => { console.log('Formulario enviado'); // Lógica de creación } });

Modal de Alerta

Modal para mostrar información importante.

showAlertModal({ title: 'Información', message: 'La plantilla se ha guardado correctamente.', buttonText: 'Entendido', onClose: () => { console.log('Modal cerrado'); } });

API Reference

UBITSModal Constructor

new UBITSModal({ id: 'modal-id', // ID único del modal title: 'Título del modal', // Título en el header content: 'Contenido HTML', // Contenido del body footerLeft: [...], // Botones del lado izquierdo footerRight: [...], // Botones del lado derecho onClose: function() {...}, // Callback al cerrar onConfirm: function() {...}, // Callback de confirmación showCloseButton: true, // Mostrar botón X closeOnBackdrop: true, // Cerrar al hacer click fuera closeOnEscape: true, // Cerrar con tecla ESC maxWidth: '440px' // Ancho máximo del modal });

Configuración de Botones

{ text: 'Texto del botón', // Texto del botón variant: 'primary', // primary, secondary, tertiary size: 'md', // sm, md, lg icon: 'far fa-save', // Icono FontAwesome onclick: 'miFuncion()', // Función onclick id: 'boton-id', // ID del botón disabled: false, // Botón deshabilitado type: 'button' // Tipo del botón }

Métodos del Modal

modal.open(); // Abrir el modal modal.close(); // Cerrar el modal modal.destroy(); // Destruir el modal modal.updateContent('Nuevo contenido'); // Actualizar contenido modal.updateTitle('Nuevo título'); // Actualizar título

UBITSModalManager

UBITSModalManager.create(options); // Crear nuevo modal UBITSModalManager.open(id); // Abrir modal por ID UBITSModalManager.close(id); // Cerrar modal por ID UBITSModalManager.destroy(id); // Destruir modal por ID UBITSModalManager.closeAll(); // Cerrar todos los modales UBITSModalManager.destroyAll(); // Destruir todos los modales

Funciones Helper

showConfirmModal()

Función helper para crear modales de confirmación rápidamente.

showConfirmModal({ title: 'Confirmar acción', message: '¿Estás seguro?', confirmText: 'Confirmar', cancelText: 'Cancelar', onConfirm: function() {...}, onCancel: function() {...}, variant: 'primary' });

showAlertModal()

Función helper para crear modales de alerta/información.

showAlertModal({ title: 'Información', message: 'Mensaje de información', buttonText: 'Aceptar', onClose: function() {...} });

showFormModal()

Función helper para crear modales con formularios.

showFormModal({ title: 'Formulario', formHTML: 'HTML del formulario', submitText: 'Guardar', cancelText: 'Cancelar', onSubmit: function() {...}, onCancel: function() {...} });
